# Wine Quality Prediction using Machine Learning

## Project Overview
This project predicts wine quality using Machine Learning techniques. 
The model analyzes different chemical properties of wine and predicts its quality score.

## Algorithms Used
- Linear Regression
- Polynomial Regression
- Support Vector Regression (SVR)

## Libraries
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Wine Quality Dataset

## Evaluation Metrics
- R² Score
- Mean Squared Error (MSE)
